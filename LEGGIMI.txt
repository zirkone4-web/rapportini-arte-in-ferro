RICOSTRUZIONE MOBILE INTEGRATA - FASE 1

Questo pacchetto:
- lavora solo sul ramo ricostruzione-mobile-integrata;

ESECUZIONE
1. Controlla in GitHub Desktop che il ramo corrente sia:
   ricostruzione-mobile-integrata
2. Estrai questo ZIP sul Desktop.
3. Avvia APPLICA_RICOSTRUZIONE_MOBILE.cmd.
4. Attendi il messaggio RICOSTRUZIONE MOBILE FASE 1 APPLICATA.
5. Torna su GitHub Desktop.
