# Arte In Ferro Mobile V2

Progetto Flutter nuovo e indipendente dalla vecchia app mobile.

Il software Windows e questa app utilizzano lo stesso progetto Supabase, quindi
condividono utenti, clienti, rapportini, presenze, mezzi, comunicazioni, foto e
firme.

La generazione delle piattaforme Android/iOS avviene in GitHub Actions: nel
repository sono conservati soltanto il codice Flutter pulito e il workflow.
